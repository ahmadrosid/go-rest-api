# Go Rest Api

Simple REST API in golang using jsonapi format.

## How to install?

Clone to your local machine :
```bash
git clone git@github.com:ahmadrosid/go-rest-api.git
cd go-rest-api
go test
```

Prepare database :
```bash
mysql -u root -psecret < data.sql
```

Compile and run :
```
go build
./go-rest-api
```
